## Variant

버튼 종류(색상)에 따른 옵션

- **base(default)**: 기본 회색 버튼
- **primary**: 브랜드 컬러 버튼
- **secondary**: 밝은 브랜드 컬러 버튼
- **text**: 텍스트만 보여지는 버튼
- **inverse**: 기본 회색의 반대 색상의 버튼
