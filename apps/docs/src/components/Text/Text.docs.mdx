## Variant

텍스트 타입(색상)에 따른 옵션

- **base(default)**: 기본 색 (`$onSurface`)
- **variant**: 변형 색 (`$onSurfaceVariant`)
- **subdued**: 약간 밝은 색 (`$onSurfaceSubdued`)
- **disabled**: disabled 된 텍스트 또는 placeholder을 위한 색 (`$onSurfaceSubdued`)
- **inverse**: 반대 색 (`$inverseOnSurface`)
- **primary**: 브랜드 색 (`$primary`)
- **error**: 에러 색 (`$error`)
