## 전체화면

모달을 전체화면으로 보여주고 싶은 경우에 옵션

**참조**

- `height`와 `paddingX` 그리고 `borderRadius`가 없어짐
- size는 주어진 옵션대로 적용이 됨 (`maxWidth`)
